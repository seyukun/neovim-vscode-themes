# neovim-vscode-themes
This repo provides an install script that easily makes neovim look and feel like vscode.

# Require packages
- jq
- git
- neovim
- nodejs
- npm

# Install
First 
```bash
curl -sL https://raw.githubusercontent.com/ES-Yukun/neovim-vscode-themes/main/install.sh | bash -
```
Second
```bash
# bash
neovim
```
```txt
:PlugInstall
```
